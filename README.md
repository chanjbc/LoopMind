## LoopMind

Fleet Optimization.

**Prerequisites:**  Node.js

## Setup

1. Clone the repo:
```bash
git clone https://github.com/chanjbc/LoopMind.git
cd LoopMind
``` 
2. Install dependencies:
```bash
npm install
```
3. Set your API key. First create `.env.local`:
```bash
touch .env.local
```
Then set `GEMINI_API_KEY` to your Gemini API key (`GEMINI_API_KEY=<API_KEY>`).

4. Run the app:
```bash
npm run dev
```
